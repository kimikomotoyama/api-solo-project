Welcome to Cookpod!
This is an API where you can create, update, delete, and list all the recipes you want in your own database. Keeping your favorite recipe notebook can be annoying, so this is where you can log all of your favorite recipes.
GET recipes/list
GET request to list all the recipes in database.
Go to page
POST recipes/
POST request to create a new recipe in database.
Recipes
  • title (string)
  • servingSize(string)
  • prepareTime(string)
  • ingredients(json)
